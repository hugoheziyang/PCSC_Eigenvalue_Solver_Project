\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classMatrix}{typename}} \mbox{\hyperlink{classMatrix}{T}}$>$\newline
class Box\+Typed$<$ T $>$}
Concrete \doxylink{classBox}{Box} that stores an object of type T by value. 

Box\+Typed$<$\+T$>$ implements the \doxylink{classBox}{Box} interface for a specific type T\+:
\begin{DoxyItemize}
\item it stores a T inside,
\item reports T\textquotesingle{}s type via \doxylink{classBoxTyped_a3d34cd67cda136292a6f9194a5cce8ae}{type()},
\item and implements \doxylink{classBoxTyped_a9d6e3a343bd01252f6f221f414e686db}{clone()} by allocating a new Box\+Typed$<$\+T$>$ with a copy of T.
\end{DoxyItemize}

This is the building block used by type-\/erased wrappers (\doxylink{classMatrix_a12d35ef1824cd847868a8703035b0d5d}{e.\+g}. \doxylink{classMatrix}{Matrix}) to hold different concrete types behind a single Box\texorpdfstring{$\ast$}{*} / \doxylink{classMatrix_a12d35ef1824cd847868a8703035b0d5d}{std\+::unique\+\_\+ptr$<$\+Box$>$}.
