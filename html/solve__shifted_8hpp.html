Solve the shifted system (A - λ I) x = b.

This function works with both dense and sparse matrices stored inside the Matrix wrapper:

  • If A_wrapped.isDense() is true, it expects the underlying type to be Eigen::Matrix<Scalar, Dynamic, Dynamic> and uses Eigen::PartialPivLU for solving.
  • If A_wrapped.isDense() is false, it expects the underlying type to be Eigen::SparseMatrix<Scalar> (the canonical sparse type used by Matrix) and uses Eigen::SparseLU.

In both cases it forms 

 M = A - λ I

and then solves M x = b.
